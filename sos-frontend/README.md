# SOS Frontend

### Setup Instructions
1. Run Front end Setup:

```
cd sos-frontend
pnpm install
npm i -g pnpm # if pnpm is not installed

```
2. Start the development server:

```
cd sos-backend
npm install
npx json-server db.json --port 3000

### Adding Shadcn Components.

```
pnpm dlx shadcn@latest add button
```
